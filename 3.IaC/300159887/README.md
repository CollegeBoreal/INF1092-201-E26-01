Nous avons observé les composantes internes, notamment les processeurs, les barrettes de mémoire RAM, les ventilateurs et les disques durs. Nous avons aussi accédé aux outils de configuration et du système pour vérifier l’état du matériel. Cette activité nous a permis de découvrir les étapes de démarrage d’un serveur et l’importance de la configuration du stockage et du matériel dans un environnement informatique.
