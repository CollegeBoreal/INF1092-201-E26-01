Pour ce travail-là, j'ai démonté un serveur HP ProLiant DL380 G6 au complet, puis je l'ai remonté. J'ai commencé par sortir les deux blocs d'alimentation à l'arrière, après j'ai ouvert le châssis pour accéder à l'intérieur. J'ai retiré les disques durs SAS un par un, enlevé les ventilateurs hot-swap, et je suis descendu jusqu'aux deux processeurs que j'ai sortis avec leurs dissipateurs.
Une fois que tout était démonté, j'ai fait l'opération inverse. J'ai replacé les CPU dans leurs sockets, remis chaque ventilateur à sa position, réinstallé les disques, refermé le châssis et glissé le serveur dans la baie avec les rails. J'ai aussi branché le câble VGA pour la console et l'alimentation, puis j'ai vérifié que le serveur communiquait bien avec le switch Cisco déjà en place dans le rack.
Ce que j'ai retenu surtout, c'est qu'il faut être méthodique. Garder l'ordre des pièces en tête, ne pas forcer sur les connecteurs, et toujours manipuler les processeurs par les côtés sans toucher aux contacts dorés en dessous.
