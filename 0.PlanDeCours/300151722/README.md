Projet : Git et GitHub
Cours : INF1092-201-E26-01

Lors de cet atelier, nous avons découvert les bases de Git et de GitHub. Nous avons commencé par créer un compte GitHub, puis un dépôt (repository) personnel afin d'y enregistrer notre travail.

Par la suite, nous avons créé un fichier README.md ainsi qu'un dossier images pour y ajouter les captures d'écran demandées. Nous avons également appris à utiliser les principales commandes Git afin de suivre les modifications apportées au projet et de les enregistrer.

Enfin, nous avons téléversé les fichiers sur GitHub et vérifié que le dépôt était correctement organisé. Les captures d'écran suivantes présentent les différentes étapes réalisées durant cet atelier.
