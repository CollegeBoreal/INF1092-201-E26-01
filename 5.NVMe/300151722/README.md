Projet : 5.NVMe
Cours : INF1092-201-E26-01

Lors de cet atelier, nous avons installé un disque NVMe de 1 To sur un serveur Windows Server 2022. Nous avons d'abord vérifié que le serveur était fonctionnel et que l'adaptateur PCIe/NVMe était correctement installé.

Ensuite, nous avons inséré le disque NVMe dans l'adaptateur et démarré le serveur. Une fois Windows lancé, nous avons vérifié que le nouveau disque était détecté par le système. Nous avons ensuite initialisé et formaté le disque afin de le rendre disponible pour le stockage des données.

Finalement, nous avons confirmé que le disque était accessible dans Windows et prêt à être utilisé. Les images suivantes présentent les principales étapes réalisées durant cet atelier.
