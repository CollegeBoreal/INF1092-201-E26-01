# 300160006
  # Description
  L’atelier avait pour objectif d’augmenter la capacité de stockage d’un serveur HP ProLiant DL360 G6 en y installant un disque NVMe de 1 To sous Windows Server 2022 Datacenter. Après avoir vérifié l’assemblage matériel, le disque NVMe a été installé puis détecté correctement par le système lors du démarrage. J’ai ensuite procédé à son initialisation, à son formatage et à sa mise en service pour le stockage des données.
La configuration s’est poursuivie avec la vérification de la connexion réseau et l’activation du Bureau à distance (RDP) afin de permettre l’administration du serveur.
Les captures d’écran incluses dans le rapport illustrent les étapes essentielles : détection du disque, formatage, validation de l’espace disponible et configuration de l’accès à distance.
L’installation s’est déroulée avec succès : le serveur fonctionne correctement, le disque NVMe de 1 To est opérationnel, la connexion Ethernet est stable et l’accès à distance est activé. Tous les objectifs du laboratoire ont été atteints, démontrant une solution simple et économique pour étendre le stockage d’un serveur existant.
