     # DESCRIPTION
Lors de cet atelier, j’ai procédé à l’installation de Windows Server 2022 Datacenter sur un serveur en commençant par vérifier que tout le matériel fonctionnait correctement, notamment le CPU, la RAM, les disques, ainsi que la présence d’un clavier, d’un écran et d’une connexion réseau active. Après avoir confirmé que l’alimentation et le câble Ethernet étaient bien branchés, j’ai démarré le serveur et choisi la méthode d’installation via PXE Boot. J’ai ensuite sélectionné Windows Server 2022 Datacenter, choisi le disque d’installation et supprimé les anciennes partitions avant de lancer l’installation. Une fois les fichiers copiés et le système installé, je me suis connecté avec le compte Administrator et j’ai défini un mot de passe sécurisé (Boreal@2026). En résumé, j’ai installé Windows Server  en démarrant le serveur en PXE, en sélectionnant la langue et le clavier, en choisissant le disque de destination, puis en finalisant l’installation et la configuration du compte administrateur. À la fin du processus, le serveur était opérationnel et prêt à être utilisé.
